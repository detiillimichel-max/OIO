# OIO ONE - Interface Orgânica por Profundidade

**Autor:** Michel Detilli  
**Versão:** 1.0.0 (OIO 1.0)  
**Conceito:** Interface de profundidade sem telas iniciais estáticas.

## 🧬 DNA do Projeto
O **OIO ONE** é uma plataforma de mídia orgânica que utiliza a **Ergonomia Dinâmica**. O usuário é o centro da experiência (Camada de Identidade), navegando através de gestos em um ambiente de vidro (Glassmorphism).

## 🛠️ Tecnologias
* **Framework:** Next.js 15 (App Router)
* **Estilização:** Tailwind CSS (Dark Mode Profissional)
* **Motor de IA:** Vercel AI SDK (OIO Connect)
* **Banco de Dados:** PocketBase (Vibe Sync)
* **Hospedagem:** Vercel

## 📏 Regras de Arquitetura
1. **Camada 0 (Mídia):** Feed de vídeo infinito (70% da tela).
2. **Camada 1 (Interação):** Gaveta de vidro expansível (30% a 50%).
3. **Salto Quântico:** Navegação rápida por miniaturas (Quantum Dots).
4. **Módulos de Acoplamento:** Novos apps integrados via pulsação visual.
